## 0. 结论速览（只读这一节也够用；2026-07-14 全面刷新，覆盖 E1-E18）

**统一原理（E18）**：后月 IV = 波动率的基础比率（均值估计），前月 = 本周的具体预报；**P&L 恒等式 = −结构方向 × 预报带符号误差 + 小额 carry**。利润只来自预报出错，永不与精准预报为敌。两个极端区标出预报错误方向可预测的状态：

1. **主粮引擎——倒挂日历（E1/E11/E14）**：slope>1.05 时卖前月 ATM 跨式、买 ~2×DTE 后月同价跨式（1:1），tp50 或骑到底。付钱方 = surge 时刻的必付对冲客群；**恐惧超调赔率 66-71%**；跨五资产成立（区内 Sharpe 1.3-1.8），唯一中位为正、扣得起成本的区；同信号下比同期铁蝶 Sharpe 高 5 倍（唯一两腿跨骑倒挂斜率的结构）。**代价已定价**：兑付周（|M|≥EM 胜率 0-30%）、sell-the-fact（后月 TV 被掏空）、持续危机季连续赔付（2026 油 −559 回撤）；后月保护域仅 |M|≲1-2EM（Vega 饱和，超出稳赔 debit——封顶设计而非失灵）。
2. **副业引擎——深正挂反蝶（E9/E13/E16）**：slope<0.95 **且 MRR≥0.95** 时买前月 ATM 跨式、卖 ±1EM 翼，持有到期不止盈。身份 = **方向中性的固定成本 CTA**（买被客群牛市疏忽打折的位移；E17：大涨周 IV −5.4pp 而 |M| 不小）；**仅股指常开**（发行融资流 = 结构性埋单方），商品间歇（MRR 开闸时，GLD 2025-26 +146/周）。赔率 54%、平衡比值 0.8（翼贴水下移）；最肥矿脉 = "塌价塌到中途"（dc ∩ IV 不低：SPY 实现比值 1.40、+84/周，小样本）。翼收入贡献大部分风险调整收益（裸跨 Sharpe 塌至 0.33）；1.0×EM 是高原中心（1.25-1.5× 为合法风格拨盘）。
3. **中性区 = 空仓**：两边均值靠尾部彩票撑（top5 占 126-908%），五件套全不过关。三区制组合（真日历复验后）：SPY +10,376/Sharpe 0.78、QQQ +8,176/0.68。
4. **MRR（位移兑现率）的诚实定义（E13）**：**举证责任分配器，非预测器**（前瞻相关 ≈0 已实测）——价值 = "默认不持有"先验在负期望环境的机械价值 + 微弱时代倾斜；商品必备（GLD p=0.021）、SPY 中性、QQQ 纯保险（置换 p=0.92，常开资产无可探测）。
5. **信号几何冻结且不可迁移**：7d/14d 对唯一（5/10、10/20 死；14/28 为第二引擎候选，待过重叠持仓/成本/QQQ 一致性三关）；0.95/1.05 阈值仅在 ~2× DTE 几何有效（宽对天然更低，14/90 中位 0.908）；逐对狩猎 = 多重检验 + 事件台阶陷阱。
6. **客群指纹决定各资产引擎菜单（E17）**：SPY/QQQ corr(ret,ΔIV) −0.7（怕跌客群→两台引擎全开）；GLD ≈0（日历主粮 + 闸控反蝶）；USO −0.22 两侧客群（倒挂闪烁窗 13.5%、中位 1 周——ETF 周度 ATM 之外的账单在 skew/远月，CL FOP 首要课题）；SLV 对称微怕涨（ETF 单笔太薄）。
7. **哲学三条（E18 定稿）**：赚"不得不对冲"的成本 ∧ 必须与共识相反（两条实为一条：被迫流制造预报偏差）∧ 结构永远封顶（硬市场牌照 = 活到 surge 的资本；永不裸卖、外围长跨书是执照年费）。
8. **负知识（同样可靠）**：IV 水平闸劣于 slope 且方向反；RV/IV 与 vol 指数造不出闸（次序不可见）；5:4 日历的第 5 手 = 与外围书重复的付费保险；跨期反蝶"卖隆起后月翼"收割仅票面 2%；常态承保（软市场）GLD 全周期 −1,660；中性区日历 +2.6 = 彩票尾款非账单收入。
9. **执行纪律**：全部建议级、纸面跟踪先行、预登记停用线（实盘落入回测 bootstrap 下 5% 即降级）；报表标配五件套（均值/中位/胜率/集中度/盈亏平衡成本）+ 分布描述。

## 1. 缘起：周末 IV 假象与 λ 框架

**观察**：跨周末的组合在模拟器里"总是有利可图"；IV Term Structure 上跨周末的到期日 IV 有系统性锯齿。

**诊断**：TWS 报价按日历日年化 IV，而周末几乎不产生方差。周末躺在远期到期日的分母里，把它的报价 IV 机械拉低（公平定价下也会呈现 ~29% 的假倒挂：cal 5 天 / trad 3 天的到期日 vs cal 2 / trad 2 的到期日，√(365/219) ≈ 1.29）。模拟器用日历时钟 + 冻结 IV，则周五附近高估远腿 ~34%、周一低估 ~29%——"跨周末稳赚"的主体是钟表读数假象。

**修复（λ 框架）**：非交易日按 λ 个交易日计权。`有效DTE = 交易日 + λ×非交易日`，`有效年 = 252 + 113λ`，IV 保总方差换算。λ=1 即日历钟（TWS 原状），λ=0 即纯交易日钟。**目测校准结果 λ≈0.3 使期限结构最平滑**（校准对象是曲线平滑度而非持仓收益，属拟合市场而非拟合愿望）。年化基数常数（252/365）在比较与定价中均约掉，唯一实质参数是 λ。

已实装：IVTS 页 TD IV 列（λ 全局可调，默认 0.3）；index.html/chart_lab 模拟时间口径三档（Calendar / Trading / Weighted λ）。

## 2. 实验记录（猜测 → 设计 → 结果 → 判定）

### E1. Calendar vs 铁蝶基线（SPY）

- **猜测**：卖近买远 Calendar 与铁蝶都收 VRP；哪个更优取决于 regime。
- **设计**：每周最后交易日入场；空前月（7±2 DTE）ATM 跨式；Calendar 腿 = 买同行权价 ~2×DTE 跨式（净支出）；铁蝶腿 = 买同期 ±EM 翅膀（EM = ATM 跨式价，净收入）。持有到前月到期：前月与翅膀按内在价值结算，Calendar 远腿按到期日 EOD mark 卖出（深 ITM mark 以内在价值为下限，修正 EOD 陈旧报价）。Regime = 前/后月 ATM IV 的 TD(λ=0.3) 比值，>1 为倒挂。
- **结果**（704 周）：

| Regime | n | 铁蝶均值 | Calendar 均值 | 配对差 (cal−fly) |
|---|---|---|---|---|
| 正挂 | 469 | −11.4 | −10.9 | +0.5（cal 胜 51%） |
| 倒挂 | 235 | +8.7 | **+25.3** | **+16.6**（55%） |
| ALL | 704 | −4.7 | +1.2 | — |

- **滑点敏感性**（$/股/腿）：0.02 时正挂 −19.4/−22.9，倒挂铁蝶 +0.7（归零）、**Calendar +13.3（唯一存活）**。
- **判定**：倒挂 → Calendar 确认；正挂 → "铁蝶更优"**证伪**，两者皆负，正确答案是不开仓。

### E2. Slope 分位与年度稳定性（SPY）

- 五分位配对差单调集中于深倒挂：Q5（slope 1.05~9.57，144 周）cal +35.5 / fly +13.6 / 配对 +22.0。深正挂（Q1-Q2，slope<0.95）对卖方最毒（fly −17.1 / −29.4）。
- 倒挂逐年：多数年份 cal ≥ fly；2018/2019/2021/2022 连续为负（特种武器也有连败期），2023-2026 最肥。

### E3. VRP 分解：正挂时溢价去哪了（SPY，n=671）

- **猜测（用户）**：正挂时未来 RV 会"打脸"过去 RV 并盖过 IV，所以低 IV 期卖 IV 不聪明。
- **设计**：跨式隐含逐日波动（EM/(0.7979·S·√交易日)）vs 过去 21 交易日 RV vs 交易内窗口前瞻 RV（每日 bp 口径）；同时记录价格空间的跨式损益（卖价 − |终点位移|）。
- **结果**：

| 分组 | IV_d | RV过去 | RV未来 | IV−RV未来 | 跨式$均值/中位 |
|---|---|---|---|---|---|
| 正挂 (449) | 78.8 | 84.2 | 72.1 | **+6.8bp** | **−1.0** / +38 |
| 倒挂 (222) | 126.5 | 111.5 | 114.8 | +11.7bp | +58.1 / +126 |

  正挂频率检验：IV>RV过去仅 49%；RV未来>RV过去仅 **32%**；RV未来>IV 仅 **31%**。IV 分位切片：溢价（bp）五个分位全正；价格空间 Q2（60-73bp）最毒（−26.4），Q3 最肥（+71.1）。
- **判定**：原猜测**证伪**——正挂时波动大概率继续压缩，市场已在 IV 里打了正确的折扣。**vol 口径溢价恒正**；亏损来自另一个维度：**中位 +38 / 均值 −1 的裂口 = 少数趋势周的大额位移**。核心发现：`|Σr| ≠ √(Σr²)`，两者之差（趋势项）才是主角。

### E4. 静态正 Delta 叠加

- **设计**：现有交易上叠加固定 +25/+50 股（线性可加，纯算术）。
- **结果**：组合对纯持股的**增量**不变（正挂仍 −11/−11，倒挂 cal 仍 +25.3）。表面指标改善（cal+25Δ 胜率 73%、中位 +57）全部来自 beta。
- **判定**：静态 Delta 不修复引擎；它的正确角色是基准对齐（客户以指数为基准）。期权账本必须以"对基准的增量"考核。

### E5. QQQ 复制（664 周）

- 基线：正挂 fly −19.4 / cal −23.8；倒挂 fly −0.8 / **cal +13.2**（配对 +14.0，56%）。
- VRP 分解逐字复现：正挂 IV 103.6 / RV过 107.3 / RV未 95.5（+8.1bp；价格空间 −15.4）；频率 50%/34%/34%。
- **判定**：全部机制跨标的成立。QQQ 趋势更强 → 正挂对卖方更毒。

### E6. 结构性偏心（把 Delta 长进帐篷）

- **设计**：中心上移 0.5 / 1.0 × EM（regime 打标仍用真 ATM）。
- **结果**（正挂均值）：SPY +1.0EM（入场 Δ≈+25/+19 股）：fly **+8.9** / cal **+9.5**（翻正）；倒挂 cal +37.2。QQQ +1.0EM：−10.5 / −5.2（改善未翻正）；倒挂 cal +28.5。全样本偏心 cal：SPY +18.6/周、QQQ +6.5/周。
- **判定**：偏心把趋势出血转化为漂移捕获，期权层修复到 ≈0~正；但相对等效纯 Delta 的超额 ≈ 0——是止血，不是新 alpha。

### E7. 出场规则（每日路径 mark，SPY 2695 / QQQ 2540 交易日行）

- tp25（+25% 权利金/支出即离场）为全天候最优修饰器：SPY 正挂 fly −11.4→−2.3、cal −10.9→−2.0；**QQQ 倒挂 cal +13.2→+34.1**（持有 4.9→3.7 天，胜率 69%）。SPY 倒挂 cal 用 tp50（+26.7）。
- **EOD 粒度止损完全无效**（sl100 ≈ hold）：亏损快而晚，日线拦不住。风控只能靠结构。

### E8. 周中再对中（离散对冲实验，裸跨实验室）

- **设计**：周中交易日平掉原跨式（EOD mark，内在价值下限），按新 ATM 重挂至前月到期；与持有到底配对比较；按前/后半周位移同号（趋势）/异号（反转）切分。
- **结果**（SPY 703 / QQQ 662）：

| 切分 | 持有 | 再对中 | 配对差 |
|---|---|---|---|
| SPY 趋势周 (373) | −174.4 | +30.8 | **+205** |
| SPY 反转周 (330) | +241.0 | +9.2 | −232 |
| SPY 正挂 / 倒挂 | +1.2 / +59.3 | +5.6 / +50.9 | +4.3 / −8.4 |
| QQQ 正挂 / 倒挂 | −15.1 / +53.4 | +1.5 / +21.0 | **+16.6** / −32.4 |

- **判定**：一次再对中把路径赌注（±200）压扁到 ±30，四个 regime×方向格 SPY 全正——**实证确认"持有到期跨式 = 位移交易；离散对冲即可掰回方差交易"**。规则：正挂再对中、倒挂骑到底。趋势周占 53~55%（正漂移时代周内自相关为正）。另一副产品：裸跨正挂 +1.2 vs 铁蝶 −11.4 ⇒ **翅膀税 ≈ $12.6/周 ≈ 正挂全部溢价**。

### E9. 反向铁蝶与三区制组合

- **猜测（用户）**：既然正挂卖方处处碰壁，反向操作——正挂买 ATM 跨式卖两翼（最大亏损 = 净支出，出现在钉住不动时）；倒挂只做 Calendar。
- **镜像性质**：反蝶损益 = −铁蝶损益（同组交易取负），净支出 = 铁蝶净收入（SPY 均值 ~$3-5/股）。
- **结果**：
  - 反蝶单独：SPY 正挂 +11.4（中位 +54，胜 57.6%，p5 −356）；QQQ 正挂 +19.4（胜 60.1%）。**反蝶不要提前止盈**（QQQ tp25 降均值）；Calendar 在倒挂要 tp50/tp25。
  - 双区制组合（正挂反蝶 + 倒挂 cal）：SPY +16.1/周（16 年 +11,307，Sharpe 0.58，maxDD −2,168）；QQQ tp50/tp50 +22.9/周（+15,181，Sharpe 0.83）。滑点 0.02 下 SPY +6.7、QQQ +13.5，仍活。
  - **三区制**（<0.95 反蝶 / 0.95-1.05 空仓 / >1.05 cal）：SPY 总利润持平（+11,464）、开工周 398/704、每开工周 +28.8、maxDD −1,371、**Sharpe 0.84**。QQQ 三区 +7,178 / Sharpe 0.55（见盲点 B4：QQQ 深倒挂异象）。
- **判定**：用户构想成立；三区制为纯改善（同利润、少干活 43%、回撤 −37%）。

### E10. 泛化性检验（"是不是牛市造出的假规律？"）

- **方向分解**：深正挂反蝶——SPY 上涨周 +25.4 (n=144) vs 下跌周 **+24.9** (n=114)；QQQ +35.6 / +34.3。**利润与方向无关**。
- **隐含 Delta 回归**：SPY −0.025 股、QQQ +0.021 股（≈零 beta）；控制方向后周 alpha +24.6 / +33.1。对 |位移| 的 β = +0.24——真实因子是位移。
- **pin/大位移**：pin 周（<0.5EM，~30%）均值 −168/−181；大位移周（≥1EM，~45%）+160/+169。
- **时间对半**：SPY 2010-17 +1,784（Sharpe 0.64）/ 2018-26 +9,680（0.97）；QQQ +2,072（**1.39**）/ +5,106（0.51）。四格全正，两品种两半互补——不存在"只有 2020 后成立"。
- **时代机制（|周位移|/EM，深正挂）**：SPY 0.88（2010-14，反蝶 −4.9）→ 0.92（2015-19，+1.3）→ **1.10**（2020-26，+56.7）。QQQ 三个时代 1.11/1.09/1.05，反蝶三代皆正（+8.5/+13.5/+57.7）。
- **判定**：牛市依赖**证伪**（2022 大熊年组合为正是旁证）；真实暴露 = 安静期位移定价偏差方向，**可每周观测**，无需预测。

### E11. 跨资产普适性检验（GLD / USO / SLV，2026-07-13）

- **猜测（用户）**："铁蝶赚不到 VRP 的全部、跑不赢 |M|，所以正挂反着做、倒挂卖近买远"——这个结论能否移植到商品 ETF？
- **设计**：同一脚本、同一冻结参数（2010–2026，front 7±2 DTE、back 2×、λ=0.3、区界 0.95/1.05、水位 26 样本均值 ≥0.95、滑点 0），跑数据库全部三个商品 ETF（GLD 710 / USO 682 / SLV 725 周）。Calendar 腿持有到期（未取每日 mark；tp50 在股指上只更好，此处偏保守）。
- **结果——倒挂 Calendar 腿（普适 ✅）**：区内周 Sharpe GLD **1.59** / USO **1.52** / SLV **1.81**（同区铁蝶仅 0.72/0.26/0.79）；正收益年份 12/17、11/16、12/17；剔除 2025-26 后腿利润仍 +2,464 / +1,401 / +696。与股指一致。
- **结果——深正挂反蝶腿（本身不普适 ❌）**：常开状态下剔除 2025-26 后 GLD **−959**、SLV −193、USO ≈0——利润全部来自 2025-26 金银大趋势。机制：商品深正挂区 |周位移|/EM 中位数只有 ~0.8（SPY 2020s 是 1.10）——**趋势税是股指特有的（风险溢价漂移），商品没有内建漂移**，安静期位移并不被低估。
- **结果——水位闸门救回它（普适 ✅）**：冻结的 0.95 水位一个参数不改直接套用，三个商品组合 Sharpe 全部改善：GLD 0.76→**0.87**（+6,845→+7,410）、USO 0.33→**0.72**、SLV 0.64→**0.71**。分时代验证闸门判断正确：GLD 2020-24 水位 0.78 → 该时代反蝶 −34.7/周（正确否决）；2025-26 水位 1.37 → +145.8/周（正确放行）。**水位计不是股指的补丁，而是跨资产的通用开关。**
- **结构差异（2026-07-14 修正）**：商品期限结构中位更平（slope 中位 ≈1.00 vs 股指 0.97-0.98），轻度倒挂（>1.0）更常见（39-45% vs 34.5%）；但**深度倒挂（>1.05）并不更多**：GLD 22.1% > SPY 17.8% > SLV 17.1% > QQQ 15.8% > USO 13.5%（最低）。且 USO 倒挂是闪烁不是时代：81 段中位持续 1 周、最长 4 周（股指恐慌期成季）——原 "远高于股指" 表述作废。
- **单笔经济性（$0.02/股/腿摊算）**：GLD 扣费后 +5,006 仍厚实（倒挂区 cal 均值 +33/周）；USO 扣费后 ≈ +310 归零；SLV **−502 转负**——标的价低、权利金绝对值太薄，1 手周均 $5-6 扛不住 6 腿成本。
- **品种特有风险**：USO 日历腿 2022 一年 +743 占腿利润 58%（俄乌油市）；USO 有展期/NAV 结构问题（2020 负油价、反向拆分）。
- **判定**：**"倒挂卖近买远"是跨资产结构性规律（VRP 的 vol 口径溢价 + 倒挂择时处处成立）；"正挂反着做"赚的是趋势税，只在有持续漂移的资产上成立——但水位闸门恰好度量"此资产此刻有没有位移溢价"，挂闸后三个商品全部改善且零调参。实操上商品只有 GLD 值得进观察名单。**

### E12. vol 指数能不能当闸（GVZ/OVX/CVOL 之问，2026-07-13）

- **猜测（用户）**：金银油都有波动率指数（Cboe GVZ/OVX 存活、VXSLV 已于 2022-02 退役；CME CVOL 覆盖 GC/SI/CL 的 FOP），拿它们做闸有没有意义？
- **归约**：vol 指数只提供隐含侧；用它能造的最强的闸 = RV/IV（隐含 vs 实现波动，经典 vol 口径 VRP）。给它最优条件（用我们自己链上的前月 IV 而非 30d 指数点，tenor 完美对口、周粒度），且允许**事后**在 0.70–1.15 网格里挑最优阈值，对比冻结 0.95 的 |M|/EM 水位闸（不许调参）。
- **结果**：GLD 水位闸 Sharpe **0.87** vs RV/IV 全网格最优 0.85；USO **0.72** vs 0.56；SLV 0.71 vs 名义 0.78——但那格只放行 1 周，实质是把反蝶腿关掉（纯日历腿 0.73），非有效闸。**事后最优的 RV/IV 闸仍不敌未调参的位移水位闸。**
- **机制**：RV = √(Σr²) 对**次序不敏感**，反蝶赚的恰是次序（趋势项 2ΣΣrᵢrⱼ）——RV/IV 闸结构性看不见被闸对象的利润来源（E3 的老结论换个马甲又验了一次）。真实 GVZ/CVOL 是 30 天平滑单点、与 7d 前腿 tenor 错配，只会更差。
- **数据陷阱记录**：`/v1/underlying-bars` 单次调用上限 260 根（静默截断到最近一年），首轮对比因此只在 2025-26 可算 RV、假阳性 Sharpe 0.99；分年抓取后修正。
- **判定**：**vol 指数当闸：无意义——闸需要"实现 vs 定价"的对照，指数只有定价侧；退一步用它配 RV 也造不出位移闸。真正用途是数据源而非信号：CVOL 作 GC/SI/CL FOP 接线后 TD IV 的交叉验证与历史参照；单指数订阅便宜（IVTS 有订阅上限）。slope 无法从这些指数得到（均为 30d 单点，无期限结构；仅 SPX 有 VIX9D/VIX/VIX3M/VIX6M 家族）。**

### E13. 反蝶是同期结构，slope 凭什么管它（"正挂是不是只要本 IV 够低"，2026-07-14）

- **猜测（用户）**：反蝶全在前月一个到期里，slope（前后期限关系）对它没有机械影响；深正挂条件是否可以换成"本期 IV 绝对水平够低"？
- **概念承认**：前半句完全正确——slope 不改变反蝶的 payoff，它只是**选周器**。问题归结为选周用相对期限关系还是绝对 IV 水平。
- **设计**：zone × 前月 IV 滚动百分位（104 周窗口、最少 26 观测）双维切分反蝶周均；反蝶腿单变量闸门对决（无水位、无日历腿）。
- **结果**：slope 闸 SPY Sharpe **0.62**（+24.3/周）/ QQQ **0.73**（+32.0/周）；纯 IV 百分位闸（<25% 或 <50%）0.21~0.41——**砍半**。方向还反了：深正挂 ∩ IV≥50% 的周均 SPY **+56.9** / QQQ **+70.2**，深正挂 ∩ IV 高段（>67%）SPY +84.0/周——给定深正挂，绝对 IV 不低的周才是金矿。
- **机制**：反蝶赚**相对定价错误**不是"便宜"。绝对 IV 低 = 市场很安静，而市场对真死水定价正确（钉住周磨净支出，2015-19 低 IV 时代 MRR 0.92、反蝶 ≈ 0）；肥肉是"带子还在动、但前月被锚在中期锚点之下"的外推错误形态。两信号高度相关（深正挂周 IV 百分位均值 0.29 / 倒挂周 0.74）——低 IV 只是深正挂的影子，拿影子当信号丢掉的恰是残差里最赚钱的部分。slope 自归一、跨波动时代可比；"IV 低"依赖回看窗口，2017 的低与 2022 的低不可比。
- **附带**：倒挂对反蝶是毒药（SPY −11.6/周），slope 直接排除它；QQQ 倒挂反蝶 +10.1 再证 B4 异象，规则仍按 SPY 版冻结。GLD 上 IV 闸略优于 slope 闸是 2025-26 单边行情污染百分位的伪象，GLD 反蝶本就必须过 MRR 闸。
- **判定**：**"深正挂"选入与"倒挂"排除都不可替代；"本 IV 够低"是更弱且方向错误的代理。剧本的第二重验证本来就是 MRR 水位——它直接度量"带子是否跑赢定价"，即"IV 低"这个直觉想抓而抓不准的东西。**
- **续（MRR 闸置换检验，2026-07-14）**：深正挂周内部按闸门开/否分组，反蝶均值差 + 置换 p 值：GLD **+47.6（p=0.021）**、SPY +29.8（p=0.079）、QQQ **−33.4（p=0.92，反向）**。解读：QQQ 的 MRR 三时代 1.11/1.09/1.05 从未换挡，无 regime 可测则读数为噪声；有效性精确追随"有没有东西可测"（GLD 两次换挡最显著）——**这是真实探测器的指纹而非拟合的指纹（拟合会在所有训练资产上均匀"有效"）**。剧本含义：闸门在 QQQ/NQ 上是纯保险（防未来换挡，样本内小幅拖累），规则不改。另：EV 对账（模拟器口径，today 结构）深正挂全体 −12 vs 开闸 +45，与本检验相互印证。
- **续（QQQ 被否周解剖，用户假设证实）**：被否周入场前 26 周涨幅 +16.7%（开闸周 +10.4%），错杀利润集中在上涨延续周（+59.1/周）与超级牛段（2021 年 21 周/+1,926、2025 +1,234），且被否周实际位移更大（1.83% vs 1.60%）——**动量资产上后视镜均值在加速期系统性低估前方位移（B4 的仪器学表述）**。但开闸周同样包含 2023 −1,037 等亏损段：QQQ 上闸门双向皆噪声，"错杀"不构成反用或拆闸的依据。诱惑中的"动量 override"参数**明确不加**（B6 分叉花园）；如要转正，须在纸面跟踪中以预登记方式验证。
- **续（MRR 持续性直接测量与定义改写，2026-07-14）**：后视 26 周 MRR 对未来 13 周兑现率的预测力**非常弱**——五资产相关系数 −0.13~+0.03；分带条件差仅 QQQ/GLD/SLV +0.03 方向正确，SPY/USO 反向。时代事后存在，但周度比值的尾部噪声淹没了可用的前瞻信息。**闸门价值重新归因**：闸门价值 ≈（−该资产反蝶腿无条件期望）× 否决率 ＋ 微弱持续性倾斜。GLD/商品的改善主要来自"默认不持有"先验在负期望环境的机械价值；QQQ 的拖累来自同一先验在常开资产上的保费；SPY 近中性。**MRR 闸的诚实定义：不是预测器，是举证责任分配器**——裸买位移默认关闭，账本近期兑付过才临时开仓。体系不再依赖"状态可预测"假设，只依赖两条硬事实：无条件裸买位移多数环境为负期望 + 亏损结构封顶。本条修正 §0 结论 6 与概念底座中"水位 = 每周可观测的残余风险因子"的表述强度：可观测（是）、可预测（弱）。

### E15. 跨期反蝶：买贱前月跨式、卖"隆起"后月两翼（2026-07-14）

- **猜测（用户）**：深正挂 = 前月 IV 低、后月 IV 相对隆起 → 反蝶的两翼改卖后月（±EM 同距离），收更肥的权利金。
- **设计**：新脚本 `scripts/backtest_cross_expiry_reverse_fly.py`（复用主回测入场/日历/regime 逻辑），每周同时构造同期反蝶与跨期反蝶做配对比较；跨期版出场 = front 到期日按市价买回后月翼（内在价值下限）。SPY 757 / QQQ 741 周。
- **定价机制证伪（翼的经济账）**：后月翼票面信用 ~2.9-3.2 是前月翼（~1.1-1.3）的 2.5 倍，但 front 到期买回时几乎全数退还——**实际收割仅 +0.06~0.07/股（票面的 ~2%）**，而前月翼全额归零。"隆起的 IV"卖不到：你卖的时间价值绝大部分在买回时还在。倒挂区收割额干脆为负（vol 继续升，翼买回更贵）。
- **但结构分裂判决（深正挂区）**：SPY 跨期 +21.7/周（Sharpe 0.95）**劣于**同期 +24.9（1.10）；QQQ 跨期 **+36.0（1.32）优于**同期 +28.6（1.08）。真实机制不是卖隆起，而是**软上限**：同期翼在大位移周变 ITM 硬性封顶利润，后月翼 delta<1 只软性回咬——趋势越猛的标的（QQQ，呼应 B4"万物皆趋势"）保留的尾部越多；代价是中位周更差（QQQ med +23.5 vs +37.2，theta 收割≈0）+ 持仓期 short 后月 vega + 两条裸卖后月腿的保证金形态（对"绝不无界卖 vol"的纪律是个刺）。
- **判定**：**假设的"卖隆起 IV"机制证伪；结构本身在 QQQ 上因软上限意外占优、SPY 上劣化——跨品种不一致，不入冻结剧本。**若未来想吃软上限，同期翼放宽到 1.5×EM 是更干净的等价表达（不引入后月 vega/裸腿），未测。

### E16. 翼距 ±1EM 有深意吗（宽度扫描，2026-07-14）

- **澄清**：翼距 = ATM call+put 的**总价**（= EM），不是单腿价。
- **设计**：同周对比 0.75/1.0/1.25/1.5/2.0×EM 五个宽度 + 裸多跨式极限（宽度→∞），SPY 756 / QQQ 740 周，同入场同 regime。
- **结果（深正挂区 Sharpe）**：SPY 1.03 / **1.11** / **1.12** / 1.07 / 0.93 / 裸 0.33；QQQ 1.02 / **1.08** / 1.05 / 1.06 / 0.98 / 裸 0.50。均值随宽度单调升（SPY +17→+36/周）但中位数单调降（+46→+11）、最差单调深（−475→−785）、胜率单调降（65%→51%）。
- **两个结论**：(1) **EM 缩放本身深意重大**——裸跨 Sharpe 塌到 0.33/0.50，确认 E8 的翅膀税：卖翼贡献了风险调整后收益的大头；(2) **但乘数在 1.0–1.5 之间是高原**，1EM 不是刀锋最优而是高原中心（这对过拟合担忧是好消息：参数不敏感 = 边际不是刀锋伪象）。宽翼（1.5×）= 更高期望 + 更糙的路径（低胜率、深左尾），是仓位偏好问题不是 alpha 问题。E15 想要的"软上限"用 1.5× 同期翼即可拿到（QQQ +41.7/周 vs 1.0× 的 +28.6，Sharpe 1.06≈1.08），无需引入后月裸腿。
- **判定**：**剧本维持 1.0×EM 冻结（高原中心、最佳胜率带）；1.25–1.5× 记为合法的风格拨盘，非改进**。

### E17. 客群指纹：IV 的方向性与"不得不对冲"的成本侧（2026-07-14，用户理论 + 实测）

- **猜测（用户）**：EM 的形成不是方向中性的。美股期权的主需求是 buy&hold 正股对冲——上涨时期权无关紧要（IV 跌，哪怕 |M| 不小），下跌时是救命稻草（|M| 非线性推高 IV）；贵金属与原油的客群不同，指纹应不同。策略应从"不得不临时来对冲"的成本支付侧出发设计，而非仅从统计入手。
- **实测（周收益 vs 前月 IV 变化，2010-2026）**：

| 资产 | corr(ret, ΔIV) | 大涨周(>+2%) ΔIV | 大跌周(<−2%) ΔIV | 客群读法 |
|---|---|---|---|---|
| SPY | **−0.73** | **−5.39pp** | **+6.17pp** | 单边怕跌客群（buy&hold 对冲盘） |
| QQQ | −0.66 | −3.78 | +5.60 | 同上 |
| GLD | **−0.05** | +0.76 | +1.62 | 无主导方向，IV 跟 \|M\| 走 |
| SLV | +0.04 | **+1.14** | +1.01 | 对称，微怕涨（挤仓恐惧） |
| USO | **−0.22** | −1.63 | +2.36 | 两侧客群（产方 put 为主、消费方 call 制衡） |

- **确认用户两个断言**：(1) "EM 宽度取决于过去 |M|"在美股不成立——大涨周 |M| 很大而 IV 反而 −5.4pp，方向而非位移驱动股指 IV；(2) 指纹跨资产分化恰如客群理论预言。
- **理论回报（把既有实证全部串起来）**：股指反蝶的边际 = **客群的牛市疏忽折扣**——上涨时对冲盘弃购期权、IV 塌得比实现位移快，而漂移持续兑付 → 深正挂 + 位移溢价。这解释了 E11（商品无此折扣：GLD corr≈0、无疏忽方向 → 反蝶无常开边际）、E13（深正挂 ∩ IV 不低是金矿：塌价中途而非塌到底）、QQQ 常开（融资流 + 最强疏忽折扣）。股指倒挂 ≈ 恐慌后状态（IV 只被下跌推高）→ 股指日历实质是"卖崩盘后恐慌"；油的倒挂两个方向都能触发（2026 上涨恐慌实例）。
- **判定与方向**：**"必须付钱的人"决定各资产哪台引擎有生意**。CL FOP 研究程序按成本侧重写：先测 skew 两侧（产方 put 溢价 vs 消费方 call 溢价）与远月（套保聚集 tenor）的可收割截面，ATM 周度只做 surge 窗口。设计标准升级为：客群指纹（本表）→ 谁在何时不得不付 → 用什么结构接住。

### E18. 损益 = 前月预报的带符号误差（恒等式 + 赔率倾斜，2026-07-14）

- **猜测（用户）**：两台引擎只在前月市场的 |M| 预报做错时赚钱，预报对了没道理赚钱；且后月锚点应解释为"均值估计/基础比率"（同债券与 VIX 期货期限结构，参与者同批、远端收敛长期均值），而非"更冷静的参与者"。
- **实测（按实现 |M|/EM 分桶，SPY/USO/GLD 全同型单调）**：反蝶——比值 <0.5 时胜率 **0%**（三资产皆然），>1.0 时胜率 **100%**，盈亏平衡 ≈0.8（翼收入把平衡点压到 1 以下）；日历完美镜像——<0.5 胜率 97-100%，>1.5 胜率 **0%**，平衡 ≈0.9-1.0。"预报精准"桶（0.8-1.0）：反蝶小赚翼贴水、日历 ≈ 零——**无人从市场答对中获利，恒等式确认：P&L = −结构方向 × 预报带符号误差 + 小额 carry**（carry 定义：预报完全兑现时结构自然结出的部分——反蝶 = 翼贴水归零的收入，日历 ≈ 0；事前可从价格算出，不是 alpha）。
- **非平凡部分（区的全部价值 = 赔率倾斜）**：dc 区内比值 >0.8 占 **54%**（三资产惊人一致）；bw 区内比值 <1.0 占 **66-71%**——**恐惧的超调（2:1）比疏忽的低估（54:46）更可靠**，与引擎分级（日历主粮）定量吻合。2026 油倒挂亏损复核：亏损全在 >1 桶（预报方向对、幅度仍不足，我们的亏损 = 它的残差）。
- **判定（交易哲学第二约束，用户定稿）**：短期收益方针 = "赚被迫对冲的成本" ∧ **"必须与市场共识相反"**——且两条实为一条：被迫的对冲流正是预报偏差的制造者（客群情绪把前月推离基础比率），我们站到偏差对面收取其均值回归。永远不与精准的预报为敌；只在预报极端偏离基础比率、且历史赔率 54-71% 倾斜时，收保费或买折扣。

## 3. 概念底座（防止未来自己绕回去）

- **两台引擎的一句话开仓时机（2026-07-14 定稿）**：**深正挂（反蝶）：价格先宣布了太平，而世界还在走——买被提前打折的位移**（slope<0.95 = 折扣已挂出；MRR≥0.95 = 世界还在走的凭证；方向不限，下行漂移同样兑付）。**深倒挂（日历）：价格宣布了末日，而恐惧大多超过兑付——卖被恐慌抬价的位移，用后月做再保险**（倒挂区 |M|/EM 中位 0.67-0.93 = 恐惧的溢价率）。统一式：**前月相对市场自己的锚点情绪化定价时，站到锚点那边——恐慌时卖前月，疏忽时买前月，两头都给亏损焊死上限。**（锚点的权威修正版见 E18：后月不是"更冷静的人"——参与者同批，后月是**均值估计/基础比率**（期限结构越远越收敛于长期均值，同债券/VIX 期货曲线）；前月是本周的具体预报（inside view），我们在预报极端偏离基础比率时押基础比率。）
- **三个量，三个资产**：隐含波动 IV（定价）、实现波动 √(Σr²)（累计路径变差）、终点位移 |Σr|（净方向结果）。`位移 ≠ √RV`；差 = 交叉项 2Σᵢ<ⱼrᵢrⱼ（符号抵消程度；过程层面 = 漂移² + 序列协方差）。**勘误（2026-07-16，用户）**：旧表述"对次序敏感/盲"是排列语言的误用——固定一组收益重排后 Σr 与 Σr² 均不变；正确直觉是固定幅度翻符号：RV 对符号模式盲、位移全是符号模式。全文旧称"次序项/趋势项"一律读作交叉项；各实验结论不受影响（它们测量的本就是交叉项）。跳跃不是此楔子的来源（孤立跳同灌两者）。
- **对称短权账本** = 收 VRP − 付趋势税（|·| 形式的 ERP 暴露）。正挂时税 > 租。
- **theta 加速（末日/最后一小时/ATM）是真实的，但它是 gamma 的租金价目表，不是利润表**；λ 修正不触碰 √T 非线性。
- **VRP 是保费不是套利**：达摩克利斯之剑的补偿，平均为正、左尾深负；能长期存在恰因收租者必须扛得住剑落。
- **两边同赢，标的市场付账（2026-07-14，用户表述）**：MRR 从不指控方差定价失效——卖方按方差定价且定贵了（IV>RV 恒成立），在自己的游戏里是赢家；但次序够单边时，按位移结算的裸持买方同时也是赢家。趋势税经做市商的 Delta 对冲（涨买跌卖）转嫁到标的市场，由逆势交易者支付——期权市场两个专业方都不因此流血，故无人修正。收割条件 = 裸持到期扛路径（对冲即退出赌局，E8）；位移溢价是"晕船费"，与 VRP 的"达摩克利斯保费"同构。
- **两个"谁埋单"的结构论证（2026-07-14，用户原创）**：(1) **原油对冲账单**：油价有内生均衡（OPEC 财政地板 + 通胀政治天花板），但对冲需求时间不耐（封井有成本、库欣存储有限、原油会结焦）→ 均衡标的上的对冲账单常年在付。指纹：USO 倒挂区 |M|/EM 中位 0.665（五资产最低）。**但中性区日历 +2.6/周经解剖证伪了"常态收账单"读法**：中位 −3.0、胜率 45.4%、前 5 周占利润 126%、赢家年全是油市动荡年（2021/2022/2026）、盈亏平衡滑点仅 $0.0043——它是廉价 long-vol 彩票（后月 vega 等喷发），不是保费收取器；真正的收账单结构（正蝶）全样本 −1.4/周——**7d ATM 粒度上超收被卖方尾部赔付吃回，软市场承保的温和版**。账单论证本身成立（|M|/EM 中位 0.67-0.83），但收银台不在周度平值：CL FOP 课题升级为"账单在哪个结构/tenor 露出可收割截面"（候选：生产商套保集中的远月 put/skew）；周度平值的钱仍在倒挂闪烁窗（+13.9/周 = 中性区 5 倍密度）。**五件套分区对照（USO 日历）**：dc 中位 −3.2/胜率 39.7%/top5 占 908%/BE 滑点 $0.0014；中性 −3.0/45.4%/126%/$0.0043；**倒挂 +1.5/51.1%/72%/$0.0231——唯一中位为正、胜率过半、集中度 <100%、成本可活的区**；且中性区内部 slope 对结果零信息（赢家入场 slope 均值 0.996 vs 输家 0.994），top-5 喷发周入场 slope 0.96-1.04 散布全带——**喷发不提前给 slope 信号，收紧阈值抓不到它们**。倒挂信号无争议成立，但 USO 倒挂中位 +1.5 仍薄（SPY 倒挂对照：中位 +34、胜率 58.4%、BE $0.0656）。2022 警示：均衡会被战争击穿，再保险层不可省。
- **倒挂日历亏损解剖（2026 回撤，两条通道）**：身份验证——bw 日历按 |M|/EM 分组：<1 → +32.8/周、胜率 71%；≥1 → **−31.6/周、胜率 4%**（日历 = 在 surge 价位卖位移，身份纯粹）。通道一（主）：**危机持续兑付**，实现位移打穿恐慌定价（2026-02-27 比值 4.09 → −130；06-18 比值 1.48 → −183）；通道二：**恐慌消退压扁后月腿**——surge 价买入的后月在正常化中跌价（05-22 比值 0.90 仍 −171、04-02 比值 0.95 → −84）。**方向澄清：越挂越倒帮助出场（后月 IV 升）；伤人的是"回归"（后月贬值）与"兑付"（前月赔付），持续深倒挂时代每周重进场 = 反复在峰值买后月**。2026 回撤（峰值 1,836 → 1,277）即两通道交替开火；hold-to-expiry 口径，tp50 或可减损未测。
- **倒挂区日历 vs 同期铁蝶正面对决（"日历优越性是不是很薄"，2026-07-14）**：USO bw——cal Sharpe **1.52**/最差 −183/p5 −87 vs fly 0.26/−454/−214；SPY bw——cal **1.29**/−428 vs fly 0.30/−836；大位移周（|M|≥EM）cal 均值约为 fly 的一半亏损（USO −31.6 vs −79.5，SPY −149 vs −274）；2026 四个关键周 cal 全胜（含 −130 vs −263 的 02-27）。**但中位数上 fly 反而略优**（USO +7.5 vs +1.5：钉住周翼归零、无后月可泄气）——fly 常赢小钱、cal 赢关键周。结构根源：**日历是唯一两腿跨在倒挂斜率两侧的结构**（卖贵前月、买贱后月，直接收割斜率异常），蝶/秃鹰两腿同月只收割水平并全额暴露宽度尾部；日历尾账单预付且固定（debit）。Short vol 结构的评分标准在坏周不在好周——优越性不薄，它恰好长在该长的地方。
- **腿级解剖与 Vega 饱和（02-27 vs 05-22）**：02-27（+33% 位移）后月完整兜住内在（单腿 +1,892），亏损 = 时间价值机械蒸发（深 ITM 跨式 TV 8.72→0.88，**delta≈1 后 Vega 已死，与 IV 无关**），净损 −129 ≈ −debit + 残余 TV——**这是日历封顶亏损按设计触底，不是对冲失灵**。05-22 才是"大动 + IV 泄气"双杀：位移中等（比值 0.90）、前腿小赚，但峰值恐慌价买入的后月 TV 17.40→2.42（对照同等 ITM 的赢家 04-24 还剩 5.31）——sell-the-fact 形态。**登记的心智模型边界：期权保护力 = 时间价值，随 moneyness 衰减归零；后月腿的保护域仅 |M| ≲ 1~2 EM。三种结局：位移小 TV 保住（赢）／位移巨大 Vega 饱和稳赔 debit（封顶）／位移中等 + 恐慌消退 TV 被掏空（最疼）。**(2) **美股发行流**：企业卖股融资是资本市场的存在方式（否则无 IPO），Gamma 对冲者追买时对面是 IPO/增发/解禁的永续融资流——位移买方的埋单人是付融资成本的结构性主体，非（仅）周期性网格盘；此即 QQQ MRR 从不换挡的经济学。**策略设计标准：说得出谁埋单才可信；MRR 举证责任仅是风控认识论，不是利润论证。**
- **硬市场牌照（2026-07-14，用户点破 + 产能顺周期补全）**：期权承保"牌照"在软市场（正挂平价期）泛滥，竞争把常态保费压至精算价以下——所以必须择时卖。倒挂超额保费未被抢光的原因：**承保资本顺周期**——保证金螺旋、VaR 收紧、上轮清洗（2018 XIV）让多数卖方在需求尖峰时刻恰好丧失出手能力。真正稀缺的牌照 = 活到 surge 且结构上能出手的资本。**永不裸卖 / 后月与翅膀封顶 / 外围双侧长跨**由此从"防守成本"重定价为"硬市场承保执照年费"：冲击周别人被抬出场，我们翻保费表。
- **可持续承保三条件（2026-07-14）**：效用买方存在 + 保费在赔付尖峰同步上浮（surge pricing）+ 再保险。倒挂日历三条全中（恐慌顶起前月 IV、|M|/EM 恰在倒挂最低、后月接尾部）；**常态承保（平静正挂期卖位移/正蝶）只满足第一条，GLD 镜像实测全周期 ≈ −1,660**：2020-24 每周 +34.7 的承保利润被 2025-26 换挡后 18 周 × −145.8 一次赔穿——保险公司死于连续赔付率超 100%，不是单次大赔付。"有人一直买"≠"随时可承保"；承保只在 surge 时刻做，这就是日历引擎的全部定义。
- **反蝶的正式身份（用户定名）**："在趋势里做损失有上限的 long displacement"= **方向中性的趋势跟随，期权只是载体**。CTA 暗付 whipsaw 当期权费，反蝶明付封顶净支出；只在期权市场对趋势暴露打折时才买（zone+MRR 就在筛这个折扣）。对标基准是 CTA 不是收租策略：震荡时代成串小额流血是日常而非异常；与正 Delta 底仓（方向趋势）、外围长跨（尾部保险）三者互不替代。
- **引擎分级原则（2026-07-14 结案）**：边际的硬度 = 对手方持续付钱的理由的硬度。**日历 = 保险费**（恐慌买方知道自己在付钱、为效用付钱，全员理性世界照样存在，恐惧可再生）→ 主粮：倒挂即开工、无闸门、标准仓位。**反蝶 = 会战战利品**（逆势流动性方不认为自己在付钱、震荡时代确实在收钱，天平会摆）→ 战利品：默认关闭、账本举证、封顶小注。验证成本与边际软度成正比（日历零质疑过五资产，反蝶经三轮认识论抢救）非巧合。仓位层（B7）应继承此分级，两引擎不同权。
- **次序 regime 的结构基础（用户还原 + Gamma 回路）**：标的市场的真实战争是动量流 vs 均值回归流动性（后者非"傻瓜"，震荡市里是赢家），裸持跨式买方 = 动量方的隐形盟友；且全街净空 Gamma 时对冲流追涨杀跌**放大**趋势、净多 Gamma 时压制趋势——头寸生态以年计换代，故次序 regime 事后成"时代"、周度不可测。MRR = 这场战争最近战况的粗糙账本；闸门据此只做举证责任分配，不做预测（见 E13 定义改写）。规则存续的三连问：依赖的结构事实（头寸生态力量对比）/断裂察觉方式（账本转负 + 预登记停用线）/察觉期代价（封顶净支出 × 半窗口）。
- **Skew 备注**：美股指数 OTM put 高地 / OTM call 洼地，对称翅膀一买一卖近似互抵（put 侧富余略胜）；GLD 类对称微笑品种性质不同。

## 附录 A. 公式

- TD IV：`iv_td = iv_cal × √( (calDTE/365) / (effDTE/effYear) )`，`effDTE = tradDTE + λ(calDTE−tradDTE)`，`effYear = 252+113λ`，λ 冻结 0.3（信号用）。
- EM（Expected Move）= 前月 ATM 跨式 mark ≈ 0.7979·S·σ·√T。
- **MRR（位移兑现率，Move Realization Ratio）**：`ratio = |ΔS| / (EM₀·√(间隔/前月DTE))`，滚动均值（26 个观测，最少 8 个）。这就是全文所称"水位计"的正式名字；≥0.95 放行反蝶（"MRR 水位闸"），<0.95 停正挂腿，样本不足 fail-closed。命名刻意避开 realized/implied 字样以区别于 RV/IV（E12 已证两者不同：MRR 对次序敏感，RV 不敏感）。
- 三区：slope = TD IV(≈7DTE) / TD IV(≈14DTE)；<0.95 反蝶持有到期；0.95~1.05 空仓；>1.05 Calendar（tp50）。
