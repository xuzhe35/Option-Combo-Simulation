# VRP 研究备忘录：从周末 IV 假象到三区制位移策略

**定稿日期**：2026-07-12
**数据**：SPY 期权链 2010-03 ~ 2026-06（704 个交易周）、QQQ 2011-06 ~ 2026-06（664 周），EOD 报价，来自 options-chain-service（Options DB workspace，`chain_server.py`，默认 `http://127.0.0.1:8750`）

## 0. 结论速览（只读这一节也够用）

1. **VRP（方差风险溢价）真实存在且从未消失**：隐含方差恒高于随后实现方差 +7~12bp/日，双标的、全 regime、全 IV 分位成立。
2. **但持有到期的对称短权（跨式/铁蝶）卖的不是方差，是终点位移** `|Σr|`。位移与方差之差 = 趋势项。路径敞口（±200/周）是 VRP（+10~20/周）的 10~20 倍。
3. **深正挂（TD slope < 0.95）时市场系统性低估周位移**（SPY 2020s：实现/定价位移比 1.10）→ **反向铁蝶（买 ATM 跨式、卖 ±EM 翅膀）有利可图**，且与市场方向无关（隐含 Delta ≈ 0）。
4. **倒挂（TD slope > 1.05）时卖方溢价足够厚** → **跨期 Calendar（卖近月 ATM 跨式、买 ~2×DTE 远月）**是最优卖方结构，骑到底或 +50% 止盈。
5. **中间带（0.95~1.05）两边都稀薄 → 空仓**。三区制：同样的总利润、57% 的开工时间、回撤 −37%、Sharpe 0.58→0.84（SPY）。
6. 残余风险因子唯一且**可每周观测**：安静期的 |周位移|/EM 水位。跌回 0.9 一带（2010-14 式环境）则正挂腿停工。

## 1. 缘起：周末 IV 假象与 λ 框架

**观察**：跨周末的组合在模拟器里"总是有利可图"；IV Term Structure 上跨周末的到期日 IV 有系统性锯齿。

**诊断**：TWS 报价按日历日年化 IV，而周末几乎不产生方差。周末躺在远期到期日的分母里，把它的报价 IV 机械拉低（公平定价下也会呈现 ~29% 的假倒挂：cal 5 天 / trad 3 天的到期日 vs cal 2 / trad 2 的到期日，√(365/219) ≈ 1.29）。模拟器用日历时钟 + 冻结 IV，则周五附近高估远腿 ~34%、周一低估 ~29%——"跨周末稳赚"的主体是钟表读数假象。

**修复（λ 框架）**：非交易日按 λ 个交易日计权。`有效DTE = 交易日 + λ×非交易日`，`有效年 = 252 + 113λ`，IV 保总方差换算。λ=1 即日历钟（TWS 原状），λ=0 即纯交易日钟。**目测校准结果 λ≈0.3 使期限结构最平滑**（校准对象是曲线平滑度而非持仓收益，属拟合市场而非拟合愿望）。年化基数常数（252/365）在比较与定价中均约掉，唯一实质参数是 λ。

已实装：IVTS 页 TD IV 列（λ 全局可调，默认 0.3）；index.html/chart_lab 模拟时间口径三档（Calendar / Trading / Weighted λ）。

## 2. 实验记录（猜测 → 设计 → 结果 → 判定）

### E1. Calendar vs 铁蝶基线（SPY）

- **猜测**：卖近买远 Calendar 与铁蝶都收 VRP；哪个更优取决于 regime。
- **设计**：每周最后交易日入场；空前月（7±2 DTE）ATM 跨式；Calendar 腿 = 买同行权价 ~2×DTE 跨式（净支出）；铁蝶腿 = 买同期 ±EM 翅膀（EM = ATM 跨式价，净收入）。持有到前月到期：前月与翅膀按内在价值结算，Calendar 远腿按到期日 EOD mark 卖出（深 ITM mark 以内在价值为下限，修正 EOD 陈旧报价）。Regime = 前/后月 ATM IV 的 TD(λ=0.3) 比值，>1 为倒挂。
- **结果**（704 周）：

| Regime | n | 铁蝶均值 | Calendar 均值 | 配对差 (cal−fly) |
|---|---|---|---|---|
| 正挂 | 469 | −11.4 | −10.9 | +0.5（cal 胜 51%） |
| 倒挂 | 235 | +8.7 | **+25.3** | **+16.6**（55%） |
| ALL | 704 | −4.7 | +1.2 | — |

- **滑点敏感性**（$/股/腿）：0.02 时正挂 −19.4/−22.9，倒挂铁蝶 +0.7（归零）、**Calendar +13.3（唯一存活）**。
- **判定**：倒挂 → Calendar 确认；正挂 → "铁蝶更优"**证伪**，两者皆负，正确答案是不开仓。

### E2. Slope 分位与年度稳定性（SPY）

- 五分位配对差单调集中于深倒挂：Q5（slope 1.05~9.57，144 周）cal +35.5 / fly +13.6 / 配对 +22.0。深正挂（Q1-Q2，slope<0.95）对卖方最毒（fly −17.1 / −29.4）。
- 倒挂逐年：多数年份 cal ≥ fly；2018/2019/2021/2022 连续为负（特种武器也有连败期），2023-2026 最肥。

### E3. VRP 分解：正挂时溢价去哪了（SPY，n=671）

- **猜测（用户）**：正挂时未来 RV 会"打脸"过去 RV 并盖过 IV，所以低 IV 期卖 IV 不聪明。
- **设计**：跨式隐含逐日波动（EM/(0.7979·S·√交易日)）vs 过去 21 交易日 RV vs 交易内窗口前瞻 RV（每日 bp 口径）；同时记录价格空间的跨式损益（卖价 − |终点位移|）。
- **结果**：

| 分组 | IV_d | RV过去 | RV未来 | IV−RV未来 | 跨式$均值/中位 |
|---|---|---|---|---|---|
| 正挂 (449) | 78.8 | 84.2 | 72.1 | **+6.8bp** | **−1.0** / +38 |
| 倒挂 (222) | 126.5 | 111.5 | 114.8 | +11.7bp | +58.1 / +126 |

  正挂频率检验：IV>RV过去仅 49%；RV未来>RV过去仅 **32%**；RV未来>IV 仅 **31%**。IV 分位切片：溢价（bp）五个分位全正；价格空间 Q2（60-73bp）最毒（−26.4），Q3 最肥（+71.1）。
- **判定**：原猜测**证伪**——正挂时波动大概率继续压缩，市场已在 IV 里打了正确的折扣。**vol 口径溢价恒正**；亏损来自另一个维度：**中位 +38 / 均值 −1 的裂口 = 少数趋势周的大额位移**。核心发现：`|Σr| ≠ √(Σr²)`，两者之差（趋势项）才是主角。

### E4. 静态正 Delta 叠加

- **设计**：现有交易上叠加固定 +25/+50 股（线性可加，纯算术）。
- **结果**：组合对纯持股的**增量**不变（正挂仍 −11/−11，倒挂 cal 仍 +25.3）。表面指标改善（cal+25Δ 胜率 73%、中位 +57）全部来自 beta。
- **判定**：静态 Delta 不修复引擎；它的正确角色是基准对齐（客户以指数为基准）。期权账本必须以"对基准的增量"考核。

### E5. QQQ 复制（664 周）

- 基线：正挂 fly −19.4 / cal −23.8；倒挂 fly −0.8 / **cal +13.2**（配对 +14.0，56%）。
- VRP 分解逐字复现：正挂 IV 103.6 / RV过 107.3 / RV未 95.5（+8.1bp；价格空间 −15.4）；频率 50%/34%/34%。
- **判定**：全部机制跨标的成立。QQQ 趋势更强 → 正挂对卖方更毒。

### E6. 结构性偏心（把 Delta 长进帐篷）

- **设计**：中心上移 0.5 / 1.0 × EM（regime 打标仍用真 ATM）。
- **结果**（正挂均值）：SPY +1.0EM（入场 Δ≈+25/+19 股）：fly **+8.9** / cal **+9.5**（翻正）；倒挂 cal +37.2。QQQ +1.0EM：−10.5 / −5.2（改善未翻正）；倒挂 cal +28.5。全样本偏心 cal：SPY +18.6/周、QQQ +6.5/周。
- **判定**：偏心把趋势出血转化为漂移捕获，期权层修复到 ≈0~正；但相对等效纯 Delta 的超额 ≈ 0——是止血，不是新 alpha。

### E7. 出场规则（每日路径 mark，SPY 2695 / QQQ 2540 交易日行）

- tp25（+25% 权利金/支出即离场）为全天候最优修饰器：SPY 正挂 fly −11.4→−2.3、cal −10.9→−2.0；**QQQ 倒挂 cal +13.2→+34.1**（持有 4.9→3.7 天，胜率 69%）。SPY 倒挂 cal 用 tp50（+26.7）。
- **EOD 粒度止损完全无效**（sl100 ≈ hold）：亏损快而晚，日线拦不住。风控只能靠结构。

### E8. 周中再对中（离散对冲实验，裸跨实验室）

- **设计**：周中交易日平掉原跨式（EOD mark，内在价值下限），按新 ATM 重挂至前月到期；与持有到底配对比较；按前/后半周位移同号（趋势）/异号（反转）切分。
- **结果**（SPY 703 / QQQ 662）：

| 切分 | 持有 | 再对中 | 配对差 |
|---|---|---|---|
| SPY 趋势周 (373) | −174.4 | +30.8 | **+205** |
| SPY 反转周 (330) | +241.0 | +9.2 | −232 |
| SPY 正挂 / 倒挂 | +1.2 / +59.3 | +5.6 / +50.9 | +4.3 / −8.4 |
| QQQ 正挂 / 倒挂 | −15.1 / +53.4 | +1.5 / +21.0 | **+16.6** / −32.4 |

- **判定**：一次再对中把路径赌注（±200）压扁到 ±30，四个 regime×方向格 SPY 全正——**实证确认"持有到期跨式 = 位移交易；离散对冲即可掰回方差交易"**。规则：正挂再对中、倒挂骑到底。趋势周占 53~55%（正漂移时代周内自相关为正）。另一副产品：裸跨正挂 +1.2 vs 铁蝶 −11.4 ⇒ **翅膀税 ≈ $12.6/周 ≈ 正挂全部溢价**。

### E9. 反向铁蝶与三区制组合

- **猜测（用户）**：既然正挂卖方处处碰壁，反向操作——正挂买 ATM 跨式卖两翼（最大亏损 = 净支出，出现在钉住不动时）；倒挂只做 Calendar。
- **镜像性质**：反蝶损益 = −铁蝶损益（同组交易取负），净支出 = 铁蝶净收入（SPY 均值 ~$3-5/股）。
- **结果**：
  - 反蝶单独：SPY 正挂 +11.4（中位 +54，胜 57.6%，p5 −356）；QQQ 正挂 +19.4（胜 60.1%）。**反蝶不要提前止盈**（QQQ tp25 降均值）；Calendar 在倒挂要 tp50/tp25。
  - 双区制组合（正挂反蝶 + 倒挂 cal）：SPY +16.1/周（16 年 +11,307，Sharpe 0.58，maxDD −2,168）；QQQ tp50/tp50 +22.9/周（+15,181，Sharpe 0.83）。滑点 0.02 下 SPY +6.7、QQQ +13.5，仍活。
  - **三区制**（<0.95 反蝶 / 0.95-1.05 空仓 / >1.05 cal）：SPY 总利润持平（+11,464）、开工周 398/704、每开工周 +28.8、maxDD −1,371、**Sharpe 0.84**。QQQ 三区 +7,178 / Sharpe 0.55（见盲点 B4：QQQ 深倒挂异象）。
- **判定**：用户构想成立；三区制为纯改善（同利润、少干活 43%、回撤 −37%）。

### E10. 泛化性检验（"是不是牛市造出的假规律？"）

- **方向分解**：深正挂反蝶——SPY 上涨周 +25.4 (n=144) vs 下跌周 **+24.9** (n=114)；QQQ +35.6 / +34.3。**利润与方向无关**。
- **隐含 Delta 回归**：SPY −0.025 股、QQQ +0.021 股（≈零 beta）；控制方向后周 alpha +24.6 / +33.1。对 |位移| 的 β = +0.24——真实因子是位移。
- **pin/大位移**：pin 周（<0.5EM，~30%）均值 −168/−181；大位移周（≥1EM，~45%）+160/+169。
- **时间对半**：SPY 2010-17 +1,784（Sharpe 0.64）/ 2018-26 +9,680（0.97）；QQQ +2,072（**1.39**）/ +5,106（0.51）。四格全正，两品种两半互补——不存在"只有 2020 后成立"。
- **时代机制（|周位移|/EM，深正挂）**：SPY 0.88（2010-14，反蝶 −4.9）→ 0.92（2015-19，+1.3）→ **1.10**（2020-26，+56.7）。QQQ 三个时代 1.11/1.09/1.05，反蝶三代皆正（+8.5/+13.5/+57.7）。
- **判定**：牛市依赖**证伪**（2022 大熊年组合为正是旁证）；真实暴露 = 安静期位移定价偏差方向，**可每周观测**，无需预测。

## 3. 概念底座（防止未来自己绕回去）

- **三个量，三个资产**：隐含波动 IV（定价）、实现波动 √(Σr²)（对次序不敏感）、终点位移 |Σr|（对次序极敏感）。`位移 ≠ √RV`；差 = 趋势项。
- **对称短权账本** = 收 VRP − 付趋势税（|·| 形式的 ERP 暴露）。正挂时税 > 租。
- **theta 加速（末日/最后一小时/ATM）是真实的，但它是 gamma 的租金价目表，不是利润表**；λ 修正不触碰 √T 非线性。
- **VRP 是保费不是套利**：达摩克利斯之剑的补偿，平均为正、左尾深负；能长期存在恰因收租者必须扛得住剑落。
- **Skew 备注**：美股指数 OTM put 高地 / OTM call 洼地，对称翅膀一买一卖近似互抵（put 侧富余略胜）；GLD 类对称微笑品种性质不同。

## 附录 A. 公式

- TD IV：`iv_td = iv_cal × √( (calDTE/365) / (effDTE/effYear) )`，`effDTE = tradDTE + λ(calDTE−tradDTE)`，`effYear = 252+113λ`，λ 冻结 0.3（信号用）。
- EM（Expected Move）= 前月 ATM 跨式 mark ≈ 0.7979·S·σ·√T。
- 水位计：`ratio = |ΔS| / (EM₀·√(间隔/前月DTE))`，滚动均值（26 个观测），<0.95 停正挂腿。
- 三区：slope = TD IV(≈7DTE) / TD IV(≈14DTE)；<0.95 反蝶持有到期；0.95~1.05 空仓；>1.05 Calendar（tp50）。
