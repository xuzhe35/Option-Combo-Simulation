**定稿**：2026-07-16 · **地位**：`VRP_RESEARCH_MEMO.md` 是实验室记录本（全部实验数据、E1–E18）；本文是蒸馏后的原理、剧本与经验总账。两者冲突时以 memo 的数据为准、以本文的原则为纲。
**当前状态**：QQQ 深正挂反蝶已于 2026-07-15 进入最小仓位实盘试验；预登记停用线生效（见 §6）。

## 1. 第一性原理（六条，全部经过实证或反向证伪加固）

**P1. 定价与结算的错位是一切利润的地基。**
期权按方差定价（实现方差衡量累计路径变差），持有到期的对称结构按终点位移结算（|Σr|，衡量净方向结果）。两者之差是**交叉项（序列依赖项）**：`(Σr)² = Σr² + 2Σᵢ<ⱼ rᵢrⱼ`，由符号抵消程度决定，过程层面 = 漂移² + 序列协方差。**注意（2026-07-16 用户勘误）**：对固定的一组收益，Σr、Σr²、∏(1+r) 全部排列不变——"对次序敏感/盲"是错误的排列语言；正确直觉是"固定各日幅度、只翻符号：Σr² 不动、|Σr| 天差地别"——**RV 对符号模式盲，位移全是符号模式**（跳跃不属于此楔子：孤立跳同灌两者，仅经跳跃聚集间接进入）。这道缝**无人有动机修复**：做市商 Delta 对冲后把交叉项从自己账本里消掉（对冲=退出这场赌局，E8 实证），方差卖方在自己的游戏里恒赢（IV>RV 从未失效，E3）。趋势税经对冲流转嫁到标的市场，由逆势流动性方支付。→ 两边同赢、标的市场埋单，所以溢价可以在完全公开的情况下长存。

**P2. 损益恒等式：利润 = 前月预报的带符号误差。**
P&L = −结构方向 ×（实现位移 − 定价位移）+ 小额 carry（预报精准时的翼贴水/归零项）。**无人从市场答对中获利**（E18 分桶：比值 0.8–1.0 格两个结构都≈零）。区的全部价值是赔率倾斜：深倒挂里恐惧超调占 66–71%，深正挂里疏忽低估占 54%——恐惧比疏忽可靠一倍。

**P3. 锚点 = 基础比率，不是"更冷静的人"。**
期限结构越远越收敛于长期均值（同债券曲线/VIX 期货，参与者同批、无渗透门槛）。前月是本周的具体预报（inside view），后月是基础比率（outside view）。剧本 = **在具体预报极端偏离基础比率时，系统性押基础比率**。

**P4. 谁埋单决定边际的硬度（引擎分级原则）。**
- **保险费**（倒挂日历）：恐慌对冲者知道自己在付钱、为效用付钱——全员理性世界照样存在，恐惧可再生 → 结构性收入，主粮；
- **战利品**（深正挂反蝶）：逆势流动性方不认为自己在付钱、震荡时代确实在收钱 → 周期性收入，副业。
客群指纹可测（E17，corr(周收益, ΔIV)）：SPY −0.73 / QQQ −0.66（单边怕跌 → 两台引擎全开）；GLD ≈0（IV 跟 |M| 走 → 日历主粮 + 闸控反蝶）；USO −0.22（两侧客群 → 仅 surge 闪烁窗）；SLV +0.04（微怕涨）。**策略设计标准：说不出谁埋单的策略不做。**

**P5. 承保产能顺周期 = 硬市场牌照。**
软市场（平价期）承保牌照泛滥，保费被竞争压至精算价下（GLD 常态卖位移全周期 −1,660）；倒挂超额保费未被抢光是因为多数卖方在需求尖峰时刻正被保证金押着。**稀缺的是活到 surge 且结构上能出手的资本**。永不裸卖、结构封顶、外围双侧长跨书——三者是执照年费，不是防守成本。可持续承保三条件：效用买方 + surge pricing + 再保险；缺一即软市场自杀。

**P6. 边际有精确的时空栖息地，不可迁移。**
时间：周节奏（7d/14d 唯一；5/10、10/20 死，边际集中在"整周"单位：周末缺口 + 周尺度趋势）；空间：曲线前端（vol-of-vol 最大处；28/56 同为 2× 几何但引擎不点火）。阈值 0.95/1.05 只在 ~2× DTE 几何有意义（宽对天然更低：SPY 14/90 中位 0.908，<0.95 是常态非信号）。

## 2. 冻结剧本（操作层，参数一律不再调）

**信号**（每周最后交易日收盘，官方交易所日历）：
TD slope = 前腿(~7DTE±2) ATM TD IV ÷ 后腿(~2×DTE) ATM TD IV，λ=0.3 冻结，未舍入值分类。

| 区 | 条件 | 动作 | 一句话时机 |
|---|---|---|---|
| 深正挂 | slope<0.95 **且** MRR≥0.95 | **反蝶**：买前月 ATM 跨式 + 卖 ±1.0×EM 翼，持有到期，**不止盈不对中** | 价格先宣布了太平，而世界还在走——买被提前打折的位移 |
| 中性 | 0.95–1.05 | **空仓** | 两台引擎的均值都是彩票尾款 |
| 深倒挂 | slope>1.05 | **日历**：卖前月 ATM 跨式 + 买 ~2×DTE 同价跨式 1:1，tp50 或骑到底，**不对中** | 价格宣布了末日，而恐惧大多超过兑付——卖 surge 保费，后月做再保险 |

**MRR（位移兑现率）**：26 周滚动均值 |ΔS|/(EM·√(gap/DTE))，最少 8 样本，fail-closed。
**诚实定义（E13 三轮拷问后）**：举证责任分配器，**不是预测器**（前瞻相关 ≈0 已实测）。价值 =（−资产反蝶无条件期望）× 否决率 + 微弱时代倾斜：商品必备（GLD 置换 p=0.021）、SPY 中性（p=0.079）、QQQ 纯保险（p=0.92，常开资产无可探测；已知误差：动量加速期读数偏低=错杀牛段）。

**引擎经济学（无摩擦基准）**：三区制 SPY +10,376/Sharpe 0.78、QQQ +8,176/0.68；倒挂日历区内 Sharpe：SPY 1.29 / USO 1.52 / GLD 1.59 / SLV 1.81；深正挂反蝶：SPY +24.9/周、QQQ +28.6/周（区内 Sharpe ~1.1）。成本后存活：SPY/QQQ/GLD；SLV 转负、USO 归零（ETF 口径）。

**结构细节**：翼距 1.0×EM 是高原中心非刀锋（E16：1.0–1.5 Sharpe 平坦；EM 缩放本身重大——裸跨 Sharpe 塌至 0.33，翼贴水占风险调整收益大头）；1.25–1.5× 为合法风格拨盘（期望+40%、路径更糙）。名义赔率 0.47:1 是针尖语言：实际（分布积分）0.82:1 × 63%，亏损周平均只亏半票（|M| 中位 0.36EM），74% 赢家顶格。

**仓位原则（B7 待细化）**：两引擎不同权（主粮 > 副业）；反蝶单注 = 封顶净支出 = 该周 whipsaw 票价；评价周期 = 完整承保周期，不按单季。

## 3. 已验证的猜想（原理 → 猜想 → 实证锚点）

1. 跨周末组合"稳赚"是日历钟假象 → λ 框架修复（λ=0.3 拟合曲线平滑度，非拟合愿望）；
2. VRP 恒正但持有到期赚不到它 → 位移≠方差，趋势税 10–20× 溢价（E3）；
3. 反蝶方向无关（涨/跌周 +25.4/+24.9，2022 为正，零隐含 Delta）→ 非牛市假规律（E10）；
4. 倒挂日历跨资产成立（五资产、11–12/17 年份为正）→ 保险费是结构性收入（E11）；
5. MRR 闸零调参迁移商品且"该失效处正确失效"（商品反蝶本无常开边际，闸门正确开关）→ 机制真实性最硬证据（E11）；
6. 深正挂 ∩ IV 不低是金矿（实现比值 1.40 vs 0.94）→ 速度差机制：塌价按资金流速度、现实按 GARCH 速度（E13 续）；美股旱地里拔的多是坑不是葱（低 IV 极端周 13/15 是暴跌）；
7. 同信号下日历比同期铁蝶 Sharpe 高 5 倍 → 唯一两腿跨骑倒挂斜率的结构；短 vol 结构评分看坏周（E14 续）；
8. 客群指纹跨资产分化如理论预言（E17）；发行融资流 = 股指位移买方的永续埋单人 = QQQ 常开的经济学；
9. 反蝶 = 方向中性的固定成本 CTA：把 whipsaw 从心理素质变成合约条款（用户定名，E16 注）。

## 4. 已推翻的猜想（负知识，每条一句死因）

1. **"IV 低时买位移更容易"**：慢慢变便宜 = 正确地便宜；纯 IV 百分位闸把 Sharpe 砍半且方向反（E13）；
2. **"RV/IV 或 vol 指数可当闸"**：RV 丢弃符号协方差信息（交叉项），看不见被闸对象的利润来源；开天眼调阈值仍输（E12）；
3. **"任何倒挂配对都值得做"**：tenor 曲线锯齿状，5/10 死、10/20 负；逐对狩猎 = 350 次多重检验 + 事件台阶陷阱（E13/E14）；
4. **"MRR 预测未来"**：前瞻相关 ≈0；降级为举证责任分配器后体系反而更稳（E13）；
5. **"卖隆起的后月翼更划算"**：票面 2.5 倍信用，买回时归还 98%——隆起卖不到手；结构分裂判决不入剧本（E15）；
6. **"商品有常年可收的对冲账单（周度平值口径）"**：中性区日历 +2.6 = 彩票尾款（中位 −3、top5 占 126%、盈亏平衡滑点 $0.0043）；账单存在但收银台在 skew/远月（待 CL 验证）；
7. **"持续买方存在 = 随时可承保"**：软市场常态卖位移 GLD 全周期 −1,660——缺 surge pricing 和再保险的保险公司死于连续赔付率超 100%；
8. **"后月腿 = 全程 vol 保护"**：保护力 = 时间价值，随 moneyness 衰减归零；保护域仅 |M|≲1–2EM（02-27 腿级解剖：Vega 饱和，−debit 是封顶设计非对冲失灵）；
9. **"买方长期亏就没有市场"**：保险市场反例——效用买方永续付费；买方有利窗口是趋势机制的产物非市场逻辑的保证（USO 反蝶 16 年无窗口，市场照样繁荣）；
10. **"5:4 日历更稳"**：第 5 手 = 周均 −$7~20 的付费保险，与外围长跨书重复（1:1 是回测口径）。

## 5. 已知失效模式与仪器误差表

- **倒挂日历两条失血通道**：兑付（|M|≥EM：胜率 4%，危机连续兑付如 2026 油）；回归（surge 价买入的后月被正常化掏空，sell-the-fact 最疼——前腿赚、后腿双倍赔）。方向澄清：**越挂越倒帮助出场**，伤人的是回归与兑付；持续深倒挂时代每周重进场 = 反复在峰值买后月（2026 回撤 −559 实录）；
- **MRR 后视镜误差**：动量加速期系统性偏低（QQQ 被否周追溯涨幅 +16.7%、实际位移更大）——已知方向为保守（错杀），不加动量 override（分叉花园）；
- **regime 换挡快于窗口**：各付半窗学费，损失结构封顶；换挡频率若高于半年一次则体系失效（已登记）；
- **仪器细节**：slope 分类必须用未舍入值；宽配对不得套用 0.95/1.05；`/v1/underlying-bars` 单次 260 根静默截断；EOD 深 ITM mark 需内在价值下限。

## 6. 认识论纪律（这套系统如何自我怀疑）

1. **报表五件套**：均值/中位/胜率/集中度（top5 占比）/盈亏平衡成本——五件不同向不成结论；分布必须给形状不只均值（原油 +2.6 彩票教训）；
2. **名义赔率 ≠ 真实赔率**：MaxProfit/MaxLoss 是针尖语言；真实赔率 = 分布在 payoff 上的积分；
3. **参数三问**：依赖的结构事实是什么 / 断裂时如何察觉 / 察觉期间代价多少——答不全的参数不进剧本；
4. **过拟合的反向检验**：真实仪器的有效性追随"有没有东西可测"（GLD 显著、QQQ 无效是加分）；拟合的划分器在所有训练资产上均匀有效；参数应坐高原不坐刀锋（E16）；
5. **冻结纪律（B3/B6）**：0.95/1.05/λ0.3/7d/1EM/26 周全部冻结；粗糙的正确好过精细的错误；每个新想法先问"这是不是分叉花园的新岔口"；
6. **预登记停用线**：实盘/纸面轨迹落入回测 bootstrap 下 5% 尾部 → 引擎降级停用。先写下什么叫失败，再开始数钱；
7. **工具的测度要认清**：模拟器无条件负期望与剧本条件正期望不矛盾（积分测度不同）；用 Regime-conditioned 开关并排读两个数字。
